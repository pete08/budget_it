BUDGET_IT App

Purpose: 
    Take back your budgeting with this easy tool that graphs past spending to analyze historical spending trends.

Steps:
1. git clone to local drive
2. open terminal and run*: 
    $ pip install -r requirements.txt
3. download your past credit card statements** in pdf format
4. input them into program's "stmts" folders
5. run "convert_pdfs_csv.py"


*if using pip as package manager
**program's unit tests created only calibrated for chase cc stmts
